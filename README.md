# 🚀 Naresh Kumar Dakua — Portfolio

A stunning, animated personal portfolio website showcasing my skills, projects, and journey as a designer & developer.

## ✨ Features

- **Cosmic Theme** — Starfield canvas, floating particles, portal wormhole effect
- **Custom Cursor** — Glowing purple dot with reactive ring
- **Loading Screen** — Animated gradient progress bar
- **Orbiting Skills** — Tech stack icons orbiting in concentric rings
- **Scroll Animations** — Staggered reveal-on-scroll effects
- **Card Spotlight** — Mouse-tracking radial glow on project cards
- **Tilt Effects** — 3D perspective tilt on stat boxes
- **Fully Responsive** — Beautiful on mobile, tablet, and desktop

## 🛠️ Tech Stack

- HTML5, CSS3, Vanilla JavaScript
- Google Fonts (Bebas Neue, Outfit, Syne)
- Canvas API for starfield animation

## 📁 Structure

```
portfolio/
├── index.html    # Main HTML structure
├── style.css     # All styles & animations
├── script.js     # Interactivity & effects
└── README.md     # This file
```

## 🚀 Quick Start

Just open `index.html` in any modern browser — no build step required!
